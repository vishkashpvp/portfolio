# Portfolio — vishkash

Personal site built with [Next.js](https://nextjs.org) (App Router), React, TypeScript, and Tailwind CSS.

## Routes

| Path        | Layout                                            |
| ----------- | ------------------------------------------------- |
| `/`         | Terminal-style shell                              |
| `/v2`       | Card sections (hero, work, internships, projects) |
| `/v1`       | Classic single-page flow                          |
| `/versions` | Pick a layout                                     |

## Scripts

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

```bash
npm run build      # production build
npm run lint       # ESLint
npm run check-types  # TypeScript
```

## Deploy

Deploy on [Vercel](https://vercel.com) or any host that supports Next.js standalone or Node.
